Logger

public class Logger

A logging class that can be told where to log to. All logs created by the SDK can be gotten and filtered through here.

Filtering

Every log call is automatically tagged with which thread it was called on (BG or UI), the function that it was called from, and the file it was called from. So if you wanted to only see logs from IdentityManager and the networking object:

Logger.shared.addTransport { print($0) } // just print all logs
Logger.shared.filterUnless(tags: ["Networking", "IdentityManager"])

Two methods exist to allow for filtering of the log stream.

  • Logger.filterUnless(tag:)
  • Logger.filterIf(tag:)
